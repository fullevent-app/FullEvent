# WideLoggerConfig

Configuration for the wide logger middleware.

## Usage

### Minimal Config

```typescript
wideLogger({
  apiKey: process.env.FULLEVENT_API_KEY!,
  service: 'checkout-api',
})
```

### Full Config

```typescript
wideLogger({
  apiKey: process.env.FULLEVENT_API_KEY!,
  service: 'checkout-api',
  environment: 'production',
  region: 'us-east-1',
  sampling: {
    defaultRate: 0.1,
    alwaysKeepErrors: true,
    slowRequestThresholdMs: 500,
  },
})
```
